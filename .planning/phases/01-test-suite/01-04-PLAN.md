Since `json_esc()` and `json_escape()` are `static` (file-local) in their respective .cpp files, we cannot call them directly from tests. **Mirror the implementation** into the test file:

```cpp
#include <catch2/catch_all.hpp>
#include <string>
#include <cstdio>

// Mirrored from src/llm/llm_client.cpp:16-37 for direct testing.
// The canonical copy lives in llm_client.cpp (and duplicated in llm_tools.cpp).
// If the implementation changes, this mirror must be updated.
namespace test_mirror {
static std::string json_esc(const std::string& s) {
    std::string out;
    out.reserve(s.size() + 16);
    for (char c : s) {
        switch (c) {
            case '"':  out += "\\\""; break;
            case '\\': out += "\\\\"; break;
            case '\n': out += "\\n";  break;
            case '\r': out += "\\r";  break;
            case '\t': out += "\\t";  break;
            default:
                if (static_cast<unsigned char>(c) < 0x20) {
                    char buf[8];
                    std::snprintf(buf, sizeof(buf), "\\u%04x", (unsigned)c);
                    out += buf;
                } else {
                    out += c;
                }
        }
    }
    return out;
}
} // namespace test_mirror
```

**Test cases:**

1. **"json_esc empty string"** — `json_esc("")` returns `""`.

2. **"json_esc no special chars"** — `json_esc("hello world")` returns `"hello world"`.

3. **"json_esc double quote"** — `json_esc("say \"hello\"")` returns `"say \\\"hello\\\""`.

4. **"json_esc backslash"** — `json_esc("path\\to\\file")` returns `"path\\\\to\\\\file"`.

5. **"json_esc newline"** — `json_esc("line1\nline2")` returns `"line1\\nline2"`.

6. **"json_esc carriage return"** — `json_esc("line1\rline2")` returns `"line1\\rline2"`.

7. **"json_esc tab"** — `json_esc("col1\tcol2")` returns `"col1\\tcol2"`.

8. **"json_esc control character NUL"** — `json_esc(std::string("\0", 1))` returns `"\\u0000"`. Use `std::string(1, '\0')` to construct.

9. **"json_esc control character BEL"** — `json_esc(std::string(1, '\x07'))` returns `"\\u0007"`.

10. **"json_esc control character 0x1F"** — `json_esc(std::string(1, '\x1f'))` returns `"\\u001f"`.

11. **"json_esc mixed special chars"** — `json_esc("key\"value\\with\nnewlines\tand\ttabs")` — verify all escapes applied correctly.

12. **"json_esc UTF-8 passthrough"** — `json_esc("héllo wörld 日本語")` — non-ASCII bytes (>= 0x80) should pass through unchanged. Verify output equals input.

13. **"json_esc long string performance"** — Create a 10,000 char string. Call json_esc. Verify it doesn't crash and output length >= input length. (Sanity check, not a performance test.)

14. **"json_esc all control chars"** — Loop through chars 0x00-0x1F (except \n, \r, \t which have named escapes). Each should produce `\\u00XX` format.
