| BSON → metric values | Untrusted BSON documents parsed into metric data |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-03-02 | Denial of Service | FtdcParser::zlib_decompress | mitigate | Cap decompression buffer growth (current impl doubles buffer, no absolute cap). Accept: FTDC files are local user files, not network input. |
| T-03-03 | Information Disclosure | MetricStore | accept | Metric data is read-only from local files, displayed to the local user. No PII, no network transmission. |
