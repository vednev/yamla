```cpp
struct MetricSeries {
    std::string path;
    std::string display_name;
    std::string unit;
    bool is_cumulative = false;
    std::vector<int64_t> timestamps_ms;
    std::vector<double> values;

    bool empty() const;

    std::unordered_map<std::string, MetricSeries> series;
    std::vector<std::string> ordered_keys;

    int64_t time_end_ms = 0;
    bool empty() const;
    MetricSeries* get(const std::string& path);
```

**Key test groups:**

1. **compute_rate tests** — Use synthetic data:
   - timestamps: {0, 1000, 2000, 3000} (ms)
   - values: {100, 200, 350, 500}
   - Expected rates: {100.0, 150.0, 150.0} (delta per second)
   - Test negative delta clamping: values {100, 200, 50, 100} → rates {100.0, 0.0, 50.0}
   - Test zero dt: timestamps {0, 0, 1000} → rate[0] should be 0.0

2. **lttb_downsample tests** — Use triangle wave pattern:
   - Create 1000-point triangle wave (values: 0→100→0→100→...)
   - Downsample to 50 points
   - Verify first index == 0, last index == 999
   - Verify peaks (index 250, 750) are preserved in output

3. **compute_window_stats tests** — Use known data:
   - timestamps: {1000, 2000, 3000, 4000, 5000}
   - values: {10.0, 20.0, 30.0, 40.0, 50.0}
   - Window [1000, 5000]: min=10, max=50, avg=30, p99≈50
   - Window [2000, 4000]: min=20, max=40, avg=30

4. **find_sample_at tests** — Binary search correctness

5. **Annotation severity verification (per D-10):**

   ```cpp
   // chart_panel_view.cpp line: if (e->severity <= Severity::Error) err_xs.push_back(ex);

   ```
