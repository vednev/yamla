**Modify `Makefile`:**

1. **Add zlib pkg-config detection.** After the existing `SSL_CFLAGS`/`SSL_LIBS` lines (around line 20-21), add:
```make
ZLIB_CFLAGS  := $(shell pkg-config --cflags zlib 2>/dev/null)
ZLIB_LIBS    := $(shell pkg-config --libs   zlib 2>/dev/null)
```

Note: On macOS, system zlib is available via `pkg-config` or falls back to `-lz`. On Linux, zlib is almost always available via system headers + `-lz`. If `pkg-config --libs zlib` returns empty, we need a fallback.

2. **Add fallback for zlib if pkg-config fails.** After the ZLIB lines:
```make
# Fallback: if pkg-config can't find zlib, use system -lz directly
ifeq ($(ZLIB_LIBS),)
  ZLIB_LIBS := -lz
endif
```

3. **Add ZLIB to the aggregated PKG_CFLAGS and PKG_LIBS.** Change the existing line:
```make
PKG_CFLAGS := $(CONAN_CFLAGS) $(SDL2_CFLAGS) $(SSL_CFLAGS)
PKG_LIBS   := $(CONAN_LIBS)   $(SDL2_LIBS)   $(SSL_LIBS)
```
to:
```make
PKG_CFLAGS := $(CONAN_CFLAGS) $(SDL2_CFLAGS) $(SSL_CFLAGS) $(ZLIB_CFLAGS)
PKG_LIBS   := $(CONAN_LIBS)   $(SDL2_LIBS)   $(SSL_LIBS)   $(ZLIB_LIBS)
```

4. **Verify SRCS auto-detection picks up FTDC sources.** The existing line `SRCS := $(shell find $(SRCDIR) -name '*.cpp')` already recursively finds all `.cpp` under `src/`, which includes `src/ftdc/*.cpp`. No change needed.

5. **Update TEST_DEP_SRCS to include FTDC sources.** The existing line:
```make
TEST_DEP_SRCS := $(filter-out src/main.cpp, $(filter-out src/ui/%, $(filter-out src/llm/llm_client.cpp, $(SRCS))))
```
This filters out `src/main.cpp`, `src/ui/*`, and `src/llm/llm_client.cpp`. Since `src/ftdc/*.cpp` is NOT in any of these exclusion patterns, it's already included. No change needed.

However, the test linking needs zlib too. The existing `TEST_LDFLAGS` line:
```make
TEST_LDFLAGS  := $(CATCH2_LIBS) $(CONAN_LIBS) $(SSL_LIBS)
```
Change to:
```make
TEST_LDFLAGS  := $(CATCH2_LIBS) $(CONAN_LIBS) $(SSL_LIBS) $(ZLIB_LIBS)
```

6. **Verify vendor paths preserved.** Ensure `VENDOR_C_SRCS` and `VENDOR_C_OBJS` for md4c remain unchanged. The httplib vendored include path (`-Ivendor/httplib`) in CXXFLAGS must remain.
